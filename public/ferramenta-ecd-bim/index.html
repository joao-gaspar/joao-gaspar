Ferramenta para Comparação do Desempenho de Entrega de Projetos ECD e BIM

Instrumento metodológico para avaliar e comparar projetos de Habitação de Interesse Social (HIS). Operacionalizável por agentes públicos e privados sem necessidade de especialização em BIM.

PARTE 1 Seleção dos Objetos

Identifica os pares de projetos ECD e BIM mais semelhantes entre si para garantir a validade da comparação de desempenho.

  • 1.1 Contemporaneidade das contratações
  • 1.2 Características das contratações
  • 1.3 Características das equipes
  • 1.4 Indicador de Grau de Esforço Projetual (IGEP)
  • 1.5 Consolidação e seleção dos pares

PARTE 2 Comparação de Indicadores

Coleta e registra os dados de entrega para calcular os 5 indicadores de desempenho e apresentar a síntese comparativa.

  • 2.1 Quantidade e tipos de documentos
  • 2.2 Tempo de entrega (meses)
  • 2.3 Documentos entregues por mês
  • 2.4 Docs entregues vs consolidados
  • 2.5 Versões por pacote de entrega
  • 2.6 Revisões por documento aprovado
  • 2.7 Síntese
Como usar: Preencha os dados dos projetos ECD e BIM nos formulários da Parte 1 para identificar os pares mais semelhantes. Em seguida, use a Parte 2 para registrar os dados de entrega e calcular os indicadores de desempenho. Clique em 📂 Carregar Exemplo para ver os dados reais da pesquisa.
SEÇÃO 1.1

Projetos e Contemporaneidade das Contratações

Cadastre os projetos ECD e BIM diretamente na tabela. Os cruzamentos mostram a diferença de anos entre as contratações — verde = mesma época, vermelho = épocas distintas.

Como usar: Clique em qualquer nome para renomear. Clique no ano (azul) para alterar. Use os botões + Adicionar nos cabeçalhos da tabela para incluir novos projetos. Os cruzamentos são calculados automaticamente.
SEÇÃO 1.2

Características das Contratações

Avalie a semelhança de legislação aplicável e dos objetos de cada par ECD/BIM. Pontue cada cruzamento de 1 a 6.

Escala de pontuação (1–6):
1 = Legislação diferente, Objetos diferentes (sem sobreposição)  ·  2 = Legislação diferente, Objetos semelhantes  ·  3 = Legislação diferente, mesmo objeto  ·  4 = Mesma legislação, Objetos diferentes  ·  5 = Mesma legislação, Objetos semelhantes  ·  6 = Mesma legislação, mesmo objeto (máxima semelhança)
SEÇÃO 1.3

Características das Equipes

Avalie a semelhança das equipes de projetistas, gerenciadora e fiscalização da prefeitura para cada par. Pontue cada tipo de equipe de 1 a 5.

Escala por tipo de equipe (1–5):
1 = Equipes diferentes de "escolas" diferentes  ·  2 = Equipes diferentes, mesma "escola"  ·  3 = Mesma equipe, membros diferentes  ·  4 = Mesma equipe, parte dos mesmos membros  ·  5 = Mesma equipe e os mesmos membros (máxima semelhança)
Pontuação máxima por cruzamento: 15 (5 × 3 equipes)
SEÇÃO 1.4

Indicador de Grau de Esforço Projetual (IGEP)

Insira o valor do IGEP de cada projeto para calcular as diferenças absolutas entre pares. Menor diferença = maior semelhança de esforço projetual.

IGEP — Indicador de Grau de Esforço Projetual
Desenvolvido nesta pesquisa com base em Andrade et al. (2024, 2025). Quantifica o grau de complexidade/esforço projetual de cada projeto. A diferença absoluta entre dois projetos indica o quão similares eles são em termos de esforço projetual.
